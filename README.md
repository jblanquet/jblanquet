## 🚀 Featured Work

### 🔍 Job Description Analyzer
**Python CLI tool for skills gap analysis**

This project analyzes job descriptions and compares them against a predefined skill set. It identifies matched and missing skills and provides a simple match score.

- Reads multiple job description files
- Normalizes text for consistent matching
- Matches skills using keyword detection
- Calculates a match score
- Handles file errors during batch processing

**Focus Areas:**  
Python • CLI Development • Text Processing • Automation • File I/O

### 🔧 Fortran → C++ System Translation
**WGU Excellence Award**

This project translates a legacy Fortran application into object-oriented C++. It keeps the original behavior, calculations, and output the same.

- Reworked procedural code into C++ classes
- Fixed build and linker issues
- Identified numerical precision differences
- Compared outputs to validate matching results

**Focus Areas:**  
C++ • OOP • Debugging • Cross-language systems • Precision handling
